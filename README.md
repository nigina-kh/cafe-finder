# Cafe Finder

A web application that helps users find nearby cafes using their current location.

# Created by Nigina Khasanova

## Features
- Detects user location
- Displays nearby cafes on Google Maps
- Shows ratings and addresses
- Provides directions
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- Google Maps JavaScript API
- Google Places API

## Project Structure
cafe-finder/
- index.html
- styles.css
- app.js
- .gitignore
- README.md

## Setup
1. Clone the repo
2. Get a Google Maps API key
3. Replace `YOUR_API_KEY` in `index.html`
4. Open `index.html` in your browser
